# Python Expense Tracker

A simple Expense Tracker project built using Python and CRUD operations.  
This project helps users manage daily expenses by adding, viewing, updating, and deleting expense records.

---

# Features

- Add Expense
- View Expenses
- Update Expenses
- Delete Expenses
- Simple Console-Based Application
- Beginner-Friendly Python Project

---

# Technologies Used

- Python

# How to Run the Project

## Step 1
Open the project folder in VS Code.

## Step 2
Open terminal in VS Code.

## Step 3
Run the following command:

```bash
python main.py
```
